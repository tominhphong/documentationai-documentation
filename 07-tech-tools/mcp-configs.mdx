## Docker MCP Gateway (UM790)

18 MCP servers chạy trên Docker, quản lý qua single gateway.

| Category | Servers |
|----------|---------|
| Search | Brave, Exa, Tavily |
| Browser | Playwright, Firecrawl |
| Data | YouTube Transcript, Maps |
| Memory | Knowledge Graph |
| Productivity | Email (send-email) |
| Media | FFmpeg, ImageMagick |
| Files | Filesystem, PDF Writer |
| Dev | Sequential Thinking, Code Mode |

**Tool prefix:** `mcp__MCP_DOCKER__`
**Find new servers:** `mcp__MCP_DOCKER__mcp-find`

## Troubleshooting MCP

| Vấn đề | Solution |
|---------|----------|
| MCP không load | Restart Claude Desktop |
| Connection timeout | Check internet / VPN |
| Tool not found | Verify config in claude_desktop_config.json |
| Permission denied | Check API key / token |
| Docker MCP down | Check Docker containers on UM790 |

## Cách Claude sử dụng trang này

Khi cần kết nối service, Claude sẽ:
1. Check MCP servers available
2. Dùng đúng MCP tool cho task
3. Troubleshoot nếu connection fails
4. Recommend thêm MCP server nếu cần capability mới
