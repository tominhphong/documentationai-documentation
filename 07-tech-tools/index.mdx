## Tech Philosophy

| Nguyên tắc | Chi tiết |
|------------|---------|
| AI-first | Dùng AI cho mọi thứ có thể: content, research, analysis |
| Automate repeatable tasks | Nếu làm 2+ lần → automate |
| LTD over subscription | Ưu tiên Lifetime Deals để giảm monthly burn |
| Free tier first | Dùng free tier trước, upgrade khi cần |
| Integration > isolation | Tools phải kết nối được với nhau |
