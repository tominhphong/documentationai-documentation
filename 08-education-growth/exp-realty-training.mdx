## CRM: HubSpot vs KV Core

| | HubSpot (Phong dùng) | KV Core (eXp cung cấp) |
|--|----------------------|------------------------|
| Cost | Free | Included with eXp |
| Interface | Modern, intuitive | Older, complex |
| Pipeline | Custom 7-stage | Standard |
| Automation | Good (free tier) | Advanced |
| Decision | **Keep HubSpot** | Use for IDX/leads only |
