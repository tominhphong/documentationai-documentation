# Neighborhood Profiles

<Callout kind="info">
  Khi khách hỏi "khu nào tốt?", câu trả lời phụ thuộc vào: ngân sách, gia đình, lifestyle, và commute. Trang này giúp match khách với khu phù hợp.
</Callout>

## Sherman, TX

### West Sherman
| | |
|--|--|
| **Giá trung bình** | $250K — $350K |
| **Loại nhà** | New construction, subdivisions mới |
| **School District** | Sherman ISD |
| **Pros** | Nhà mới, community amenities, gần US-75 |
| **Cons** | Giá cao hơn East Sherman, HOA fees |
| **Phù hợp cho** | Gia đình trẻ, first-time buyer budget $250K+ |
| **Commute to Dallas** | ~60 phút qua US-75 |

### East Sherman
| | |
|--|--|
| **Giá trung bình** | $150K — $250K |
| **Loại nhà** | Older homes, cần renovation, mixed |

| **Cons** | Nhà cũ hơn, có thể cần sửa chữa |
| **Phù hợp cho** | Budget thấp, investor, fixer-upper buyer |
| **Commute to Dallas** | ~65 phút |

### Downtown Sherman

## Denison, TX

| | |
|--|--|
| **Giá trung bình** | $200K — $320K |
| **Loại nhà** | Mix: historic downtown + new builds |

## Cách Match Khách Với Khu Vực

### Theo ngân sách
- **Dưới $200K:** East Sherman
- **$200K-$300K:** West Sherman, Downtown, Denison
- **$300K-$400K:** Howe, Van Alstyne, West Sherman (new build)
- **Trên $400K:** Van Alstyne (custom), Rural Grayson County

### Theo ưu tiên
- **Trường tốt nhất:** Howe ISD, Van Alstyne ISD
- **Commute ngắn nhất:** Van Alstyne → Howe → Sherman
- **Nhà mới nhất:** West Sherman subdivisions, Van Alstyne
- **Giá tốt nhất:** East Sherman, Denison (older areas)
- **Lake lifestyle:** Denison (gần Lake Texoma)

## Cách Claude sử dụng trang này

Khi khách hỏi "khu nào phù hợp?", Claude sẽ:
1. Hỏi ngân sách, kích thước gia đình, ưu tiên (trường/commute/giá)
2. Match với 2-3 khu vực phù hợp nhất
3. Giải thích pros/cons cụ thể
4. Đề xuất lịch xem nhà tại các khu được chọn
