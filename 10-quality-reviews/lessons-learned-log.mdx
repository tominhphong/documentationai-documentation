## Common Categories

| Category | Ví dụ bài học |
|----------|-------------|
| Negotiation | Nên counter thay vì accept first offer |
| Communication | Cần update khách thường xuyên hơn |
| Process | Book inspector ngay ngày 1, không chờ |
| Market | Khu X đang hot, nên focus marketing |
| Compliance | Luôn đưa IABS trước khi nói chuyện cụ thể |
| Tech | Tool X không hoạt động — cần backup plan |

## Cách Claude sử dụng trang này

Khi Phong review performance, Claude sẽ:
1. Tổng hợp lessons theo category
2. Highlight patterns (bài học lặp lại)
3. Generate monthly/quarterly retrospective
4. Connect lessons với actions cụ thể
